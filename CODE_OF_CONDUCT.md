# Code of Conduct

LifeOS is an open repository. Be direct, professional, and respectful.

## Expected Behavior

- Be constructive in issues, pull requests, and discussions.
- Critique code and decisions, not people.
- Assume good intent until evidence says otherwise.
- Keep security discussions private when details could harm users.

## Unacceptable Behavior

- harassment
- threats
- doxxing
- discriminatory language
- stalking
- repeated disruptive behavior
- public disclosure of sensitive security details

## Enforcement

Maintainers may remove content, close issues, block contributors, or restrict collaboration when behavior undermines the project.

## Reporting

Report conduct issues through the same private path used for security concerns, or directly to the maintainers if the issue is not security related.

## Scope

This code of conduct applies to repository issues, pull requests, reviews, discussions, and project-related communication.
